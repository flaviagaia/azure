# payment_anomaly_stream_azure

## Português
Projeto orientado a streaming para monitoramento de anomalias em pagamentos.

Serviços Azure de referência:
- `Azure Event Hubs`
- `Azure Stream Analytics`
- `Azure Machine Learning`
- `Power BI`

Objetivo:
- ingerir eventos de pagamento em tempo real
- calcular sinais de anomalia
- priorizar transações para revisão

## English
Streaming-oriented payment anomaly monitoring project using Event Hubs, Stream Analytics, Azure Machine Learning, and Power BI.
