# claims_approval_workflow_azure

## Português
Projeto de workflow de aprovação com foco em orquestração serverless.

Serviços Azure de referência:
- `Logic Apps`
- `Azure Functions`
- `Blob Storage`
- `API Management`

Objetivo:
- receber solicitações
- validar documentos
- aplicar política de aprovação
- separar autoaprovação de revisão manual

## English
Workflow-oriented claims approval project using Logic Apps, Azure Functions, Blob Storage, and API Management.
