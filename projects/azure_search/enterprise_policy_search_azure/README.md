# enterprise_policy_search_azure

## Português
Projeto de busca corporativa para políticas internas e documentação operacional.

Serviços Azure de referência:
- `Azure AI Search`
- `App Service`
- `Azure SQL Database`

Objetivo:
- indexar documentos corporativos
- recuperar políticas por similaridade
- servir a busca em uma aplicação web

## English
Enterprise policy search project using Azure AI Search, App Service, and Azure SQL Database.
